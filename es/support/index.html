Unirse mediante TestFlight

Para reportes de errores o consultas, utilice "Soporte → Reportar un problema" dentro de la aplicación siempre que sea posible.

Ventajas de contactar desde la aplicación

  • El nombre, versión y número de compilación se adjuntan automáticamente
  • La versión de iOS y el modelo del dispositivo se incluyen automáticamente
  • Esto nos ayuda a investigar problemas más rápidamente

Adjuntar una captura de pantalla también nos ayudará a responder más rápido.

Si la aplicación no se inicia

Si no puede contactarnos desde la aplicación, por favor envíe un correo a:

staff@a-soft.info

Cuando nos contacte por correo electrónico, incluya la siguiente información si es posible:

  • Nombre de la aplicación
  • Versión de la aplicación
  • Versión de iOS
  • Modelo del dispositivo
  • Descripción detallada del problema

Después de revisar su informe, normalmente respondemos en unos pocos días. Asegúrese de poder recibir correos de a-soft.info.
