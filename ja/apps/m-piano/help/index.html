M-Piano ヘルプ

ピアノコード検索・コード進行作成・メモ管理アプリ

M-Piano は、ピアノコードの検索・分析・整理を行うためのツールです。
コード検索、コード逆引き、コード進行メモ、コード譜インポートなどの機能を使って、 学習・作曲・演奏の確認を効率よく行えます。

まず最初に（重要）

  • コードの再生は「鍵盤をタップ」して行います（スピーカーではありません）。
  • スピーカーアイコンは「ミュート（消音）」の切り替えです。
  • PRO 表示は Pro版で使える機能 です（購入前に見ても誤解しないよう、ヘルプ内で明確に区別します）。

タブの説明

Forward

コード名からピアノコードを検索します。
・コードの構成音
・鍵盤ポジション
・複数のボイシング
を確認できます。

Reverse

押さえた鍵盤からコード名を逆引きします。
作曲やコード分析に便利です。
Reverse からメモへ追加することもできます。

Memo

コード進行を保存・整理します。
・コード進行メモ
・コード譜インポート
・トランスポーズ（移調）
・メール共有（.mpmemo）
・全画面表示（メモ／タップボタン）

PRO Pro版ではページ管理が可能
メモできる数は理論上無制限（1ページのコード数もPro版では実質無制限）。

Setting

アプリの設定を行います。
・コード表示範囲（表示できるタイプの範囲など）
・表記設定（♯/♭ など）
・アプリ情報 / サポート

M-Pianoの主な機能

  • コード名からピアノコードを検索（Forward）
  • 押さえた鍵盤からコード名を逆引き（Reverse）
  • コード進行をメモとして保存・整理（Memo）
  • コード進行のトランスポーズ（移調）
  • PRO コード譜インポート（貼り付け解析）
  • PRO.mpmemoファイルによるメモ共有

オンベース（PRO）

PRO オンベース（例：C/E、D/F# など）は Pro版のみ利用できます。
オンベースを選ぶと、上部の大きな鍵盤に ポジションマーク が表示され、
再生音も ベース音を含んだ形 で確認できます。

コード譜インポート

コード譜を貼り付けると、コード進行を自動解析してメモに取り込みます。
詳細は インポートの説明 を参照してください。

例：このようなコード譜を貼り付けると解析されます

      CM7  Bm7  D/E  Em
      頑張らなくてもいいいんだよ
      (ピアノのみ)

      C  G/B  Am7/D  D7
      G/D D/F#  Em  Em7  C  G/B  Am7  D

      (ミュートアルペジオ)
      G    D/F#    Em    Em7        C   C/B   Am7     D
        朝の光が まぶしすぎる日は 布団の中で 少しだけ夢を見よう
      G    D/F#    Em    Em7            C    G/B    Am7      G
         昨日の涙も うまく笑えない顔も 誰かと比べなくていいんだよ

Pro版について

無料版でも主要機能を利用できます。
Pro版では機能制限が解除され、さらに多くの機能が利用できます。
ヘルプ内では PRO マーク付きの項目が Pro版の内容です。

  • PRO メモのページ管理（複数ページで整理）
  • PRO メモ数・コード数の制限が大きく拡張
  • PRO 表示できるコードタイプが増える
    Settingでスタンダード(10) → スタンダード+(25) → すべて(85) と拡張できます
  • PRO オンベース（/C〜/B）に対応
  • PRO コード譜インポートと .mpmemo 共有

Pro版の詳細は Pro版の説明 を参照してください。
