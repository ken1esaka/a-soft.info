Features

It works like a normal calculator, but entered numbers and calculations are displayed in a visible list.

This allows you to follow the calculation process easily. Values can also be edited later if necessary.

Features Added in A-CalcM

Memory Function
The memory function makes cumulative multiplication calculations easy.

Example: calculating (250×100) + (350×100)

250×100 → [M+]
350×100 → [M+]
The result will appear at the bottom.

Text Input
Double-tap a cell to enter a short text note.

Email Export
The contents displayed on the screen can be exported by email.
You can choose between image or text format.

If you find any issues, please send a screenshot using the in-app contact feature.
