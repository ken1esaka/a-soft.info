Guitar Chord Book (Legacy Series)

Advanced guitar chord book covering almost every fingering

Instead of displaying pre-registered chord shapes, this app calculates guitar chord fingerings automatically.

It supports a variety of tunings including Drop-D and open tunings.

The app also provides reverse chord search, allowing advanced searches by specifying all strings.

The unlocked version allows searching through more than 300,000 possible fingerings, including slash chords and alternate tunings.

Memos can be backed up using iCloud, making them easy to restore on devices using the same Apple ID.

A web version is also being considered, although there is currently no specific schedule.

If you encounter any issues, please send a screenshot using the in-app contact feature.
