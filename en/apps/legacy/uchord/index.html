U-Chord

Ukulele Chord Book (Legacy Series)

A ukulele chord book designed for beginners

This chord book was developed to make ukulele playing and singing more accessible for beginners.

With simple controls you can quickly search for chords and practice while listening to the sound.

An Android version was available in the past, but currently only the iOS version is provided.

A web version is also being considered, although there is no specific schedule yet.

If you encounter any issues, please send us a screenshot using the in-app contact feature.
