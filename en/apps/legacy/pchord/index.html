P-Chord

Piano Chord Book (Legacy Series)

A piano chord book designed for beginners

Even if you cannot read sheet music, you can start playing songs by learning chords. This app was created with that simple idea in mind.

The app focuses on voicings that can be played with one hand, making it easy to accompany singing or practice chord progressions.

Saved chord progressions can also be transposed to different keys, which many users find helpful when practicing or composing.

An Android version was available in the past, but currently only the iOS version is provided.

A future web version is being considered, although there is no specific schedule yet.

If you encounter any issues, please send us a screenshot using the in-app contact feature.
