BU-Chord

Baritone Ukulele Chord Book (Legacy Series)

Chord book designed for baritone ukulele

This chord book was designed specifically for the baritone ukulele, which uses a different tuning from the standard ukulele.

The chord forms are selected with playability in mind, allowing you to practice while listening to the sound.

This app is provided as part of the legacy series.

A web version is also being considered, although there is no specific schedule yet.

If you encounter any issues, please send us a screenshot using the in-app contact feature.
