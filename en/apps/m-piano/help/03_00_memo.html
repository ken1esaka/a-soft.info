Memo

Chord Progression Memo

What is Memo

Memo is a feature for saving and organizing chord progressions.
You can store chords discovered in Forward or Reverse as memos and replay or share them later.

Screen Layout

The main elements of the Memo screen are as follows.

① Page Title
The title of the currently displayed memo page. You can edit the title directly to rename the page.

② Page Selection Button
Displays a list of saved memo pages. Selecting another page will show the memos on that page.

③ Add Page Button
Creates a new memo page. (Pro version)

④ Import Button
There are two types of import methods in Memo.

  • Chord Chart Text Import
    Paste chord chart text and import it as a chord progression.
    → Detailed explanation of the import screen
  • .mpmemo File Import
    Opening a .mpmemo file received by email or other means will import it as a memo page.

⑤ Position Keyboard
Displays the chord fingering (position) on the keyboard.

⑥ Tap Button
Tapping a chord name button plays the chord and shows the corresponding keyboard position.

⑦ Transpose Button
Transposes the chord progression in semitone steps.

⑧ Email / Share Button
Send memo contents via email or the share function. (Pro version)

With email export, you can share the memo page as a .mpmemo file.
See the following page for detailed instructions.
→ Detailed explanation of email export

⑨ Clear Button
Deletes all memos on the current page.

⑩ Edit Button
Allows editing such as reordering or deleting memos.

Full Screen Display

The chord buttons in Memo can be displayed in full screen.

This allows chords to be displayed larger during live performances or practice.

Number of Memos

The number of memos you can save depends on the app version.

  • Free version: Limited
  • Pro version: Virtually unlimited

Page Management (Pro)

In the Pro version, memos can be managed by pages.

Each page can have its own title.

Example

  • Practice
  • Live
  • Composition
  • Band Songs

Using pages helps you organize and manage chord progressions.

.mpmemo File

The .mpmemo file is a memo data file for M-Piano.

When you receive a .mpmemo file by email

Tap → M-Piano

Then

  • M-Piano launches
  • Imported into Memo

You can easily share chord progressions with friends or band members.
