What you can do with Pro / Differences from the Free Version

What is the Pro Version?

The Pro version unlocks all practical features of M-Piano, including extended chord types, Memo page management, and chord chart import.

The Pro version removes the functional limitations of M-Piano and allows you to fully use features needed for songwriting, practice, and sharing.

※ For users reading this help before purchasing, all Pro-exclusive features are summarized on this page.

Features Available in Pro ✅

  • On-bass chord display (Forward) ✅
  • Expanded chord type display ✅
  • Memo page management (multiple pages) ✅
  • Virtually unlimited number of chords per page ✅
  • Memo sharing (.mpmemo email → recipient taps → instant import into M-Piano) ✅
  • Chord chart import (paste → analyze → import into Memo) ✅
  • Tap chord names in import preview to play sound ✅

What You Can Do in the Free Version

The free version still allows basic usage.

  • Basic operation of Forward / Reverse
  • Check chords (tap keyboard to play, mute toggle)
  • Basic Memo usage (with limitations)
  • Edit and preview in the chord chart import screen (import execution is unavailable)

On-Bass Chords (Pro)

On-bass chords (e.g., C/E, C/G) are available only in the Pro version.

When an on-bass chord is selected,

  • Position markers appear on the large keyboard at the top
  • Playback includes the bass note when tapping the keyboard

This allows you to hear and use more practical chord sounds (slash chords).

→ See Forward explanation

Expanded Chord Types (Pro)

The Pro version increases the number of chord types available. In Setting → "Chord Type Display Range", you can expand the display step by step: Standard (10) → Standard+ (25) → All (85).

→ See Forward (Chord Selection)

Memo Page Management (Pro)

In the Pro version, Memo can be organized into multiple pages.

You can assign titles to pages, allowing organization such as:

  • Practice
  • Live
  • Composition
  • Band Songs

This helps manage chord progressions by purpose.

Also, the number of chords per page is virtually unlimited, allowing you to store long songs or multiple songs together.

→ See Memo explanation

Memo Sharing (Pro)

In the Pro version, Memo content can be sent by email.

When the recipient taps the attached file (.mpmemo) Tap → Open with M-Piano

  • M-Piano launches
  • The memo is imported

This makes it easy to share chord progressions with friends or band members.

→ See Memo sharing explanation

Chord Chart Import (Pro)

Paste a chord chart, automatically analyze the progression, and import it into Memo.

  • "Execute Import" = Pro version only ✅
  • Tap chord names in preview to play sound = Pro version only ✅

→ See Chord Chart Import explanation
