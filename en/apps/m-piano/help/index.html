M-Piano Help

Piano chord search, chord progression creation, and memo management app

M-Piano is a tool for searching, analyzing, and organizing piano chords.
Using features such as Chord Search, Reverse Lookup, Chord Progression Memo, and Chord Chart Import, you can efficiently support learning, composing, and checking performances.

Important First

  • Chord playback is done by tapping the keyboard (not the speaker icon).
  • The speaker icon toggles mute (sound on/off).
  • PRO indicates features available in the Pro version (clearly marked to avoid confusion before purchase).

Tabs

Forward

Search piano chords by chord name.
You can check:
• Chord tones
• Keyboard positions
• Multiple voicings

Reverse

Identify chord names from selected piano keys.
Useful for composition and chord analysis.
You can also add chords to Memo from Reverse.

Memo

Save and organize chord progressions.
• Chord progression memos
• Chord chart import
• Transpose
• Email sharing (.mpmemo)
• Full-screen display (memo / tap buttons)

PRO Page management available in Pro version
The number of memos is theoretically unlimited (the number of chords per page is also virtually unlimited in Pro).

Setting

Configure application settings.
• Chord display range (available chord types)
• Notation settings (♯ / ♭ etc.)
• App information / support

Main Features of M-Piano

  • Search piano chords by chord name (Forward)
  • Reverse lookup chord names from pressed keys (Reverse)
  • Save and organize chord progressions (Memo)
  • Transpose chord progressions
  • PRO Chord chart import (paste and analyze)
  • PRO Memo sharing via .mpmemo files

On-Bass Chords (PRO)

PRO On-bass chords (e.g., C/E, D/F#) are available only in the Pro version.
When an on-bass chord is selected, a position marker appears on the large keyboard at the top,
and playback includes the bass note.

Chord Chart Import

Paste a chord chart and M-Piano automatically analyzes the chord progression and imports it into Memo.
See Import Details for more information.

About the Pro Version

The free version already provides the main features.
The Pro version removes limitations and unlocks additional functionality.
Items marked with PRO in the help indicate Pro-only features.

  • PRO Memo page management (organize across multiple pages)
  • PRO Significantly expanded memo and chord limits
  • PRO Expanded chord type display
    Setting allows Standard (10) → Standard+ (25) → All (85)
  • PRO On-bass chords (/C–/B)
  • PRO Chord chart import and .mpmemo sharing

See Pro Version Details.
